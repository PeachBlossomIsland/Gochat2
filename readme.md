这个也是一个基于websocket的go语言写的服务器段的聊天室程序，是初学go语言和初次使用git进行的，代码质量不高，也算是一次尝试。
这个程序参考了很多网上的东西，主要是熟悉和理解语言为主的。当然，这个代码还不完善，需要更多的改进。main.go是服务器端代码，
